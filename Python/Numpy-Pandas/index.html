Pandas

  • Pandas处理三类数据结构：Series、DataFrame、Panel，它们均构建在Numpy数组之上；DataFrame是Series的容器，Panel是DataFrame的容器。

    • Series是具有均匀数据的一维数组结构，其尺寸大小不可变

    • DataFrame是一个具有异构数据的二维数组，一个表格其每一列的数据类型可以不同

  • Panel是具有异构数据的三维数据结构
